> Simple color palette (Backbone.js)

## Demo
[vovanr.github.io/color-palette](http://vovanr.github.io/color-palette)

## Development

### Install dependencies
```sh
npm i
```

### Build dist
```sh
gulp build
```

### Run example
```sh
gulp
```
Open http://localhost:8000
